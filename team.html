Swarm-in-a-Box: Multi-Robot Task Allocation for Smart Hospital Logistics

A highly scalable, autonomous fleet system utilizing MQTT and proximity bidding for optimized delivery logistics.

Launch Live Demo & Dashboard

Key Advantages of the MediFleet Swarm

True Scalability

Add 2 or 20 robots instantly. Throughput scales linearly with fleet size, thanks to identical, modular code.

Autonomous Bidding Logic

Robots self-allocate tasks using a proximity-based formula, ensuring the nearest unit executes the delivery.

Low-Cost Hardware

Built on ESP32, the system maintains reliability and robust communication without requiring expensive commercial AGVs.

System Architecture: Cloud Communication

The system operates on a reliable, low-latency foundation using the **MQTT Protocol** over Wi-Fi. A central manager publishes tasks; robots subscribe, bid, and execute missions, providing real-time status updates via JSON messaging.
